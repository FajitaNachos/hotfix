Welcome to hotfix.

Here's a couple points to help you get started.

First, click on the Elements tab and make some changes. Then come back here to see a list of all the files that you have edited. Select a repository and branch to commit to, enter a commit message, and commit. Simple, right? Side note: The full path of the file is what will be pushed to GitHub. In other words, the root of your GitHub repo should match the root of your project.

Also, due to the way message passing works in Chrome, this extension doesn't work properly when Chrome Developer Tools is docked to main window. Click the button to undock the toolbar in the bottom left corner, then restart the Developer Tools.

That's about it. Have fun and let me know if you have any problems.

In order to get started we need to authorize this extension to access your GitHub repos.
