Welcome to hotfix

Let's get started.

1. Click on the Elements tab in Chrome Dev Tools and make some changes to your site

2. Click on the hotfix panel to see a list of files you have edited.

3. Select a repository and branch to commit to.

4. Enter a commit message and commit.

Simple enough, right? Note: The full commit path is what will be pushed to GitHub. In general, the root of your GitHub repo should match the root of your project. If it doesn't, you will need to edit the commit path before pushing to GitHub.

That's about it. Have fun.
